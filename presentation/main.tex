\frametitle{Introduction}
\begin{itemize}
    \item Modern multi-thread, multi-core hardware focus on sharing computing resources among different programs.
    \item In cloud, VM and sandbox technology is used as software level security measure. But they share the same hardware.
    \item Programs running on current hardware inevitable leave fingerprints of their execution in power traces, EMI spectrum,
        caches, etc. \cite{side-channel}
    \item Attackers with knowledge of hardware can obtain these fingerprints and extract sensitive information.
\end{itemize}
\end{frame}
